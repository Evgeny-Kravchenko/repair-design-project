# Short instruction, how to install and start project

1. [install node js](https://nodejs.org/en/)
2. [install gulp](https://gulpjs.com/docs/en/getting-started/quick-start)
3. [install browser sync](https://www.browsersync.io/)
3. Clone this repository: [https://github.com/Evgeny-Kravchenko/repair-design-project](https://github.com/Evgeny-Kravchenko/repair-design-project)
4. Go to the folder repair-design-project
5. Run npm install in command line
6. Run gulp dev in command line
7. Open index.html in the folder "dist"




P.S.: If you want to see the source files then Go to the folder:  repair-design-project --> src
